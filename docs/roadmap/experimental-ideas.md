# Experimental Ideas

These ideas are intentionally not active Emberlane features:

- Search worker examples.
- File parsing workers beyond simple `.txt`/`.md` context.
- Plugin provider systems.
- SDKs.
- Dashboards.
- GCP and Azure backends.

They may be revisited after the local Ollama and AWS scale-to-zero paths are boringly reliable.

See `docs/roadmap/future-work.md` for the current public roadmap version of these ideas.
